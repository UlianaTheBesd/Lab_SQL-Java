## Лабораторная работа для предмета "Управление Данными".  

### Реализация однотабличной Базы Данных для предменой области "Социальная сеть" с GUI (Java Swing).  
Код на PostgreSQL принимает запросы от кода, написанного на Java, с использованием интерфейса JDBC.  

__1. Базовые функции (_прописано в SQL_):__  
create_db_socialnet() - Создание однотабличной базы данных (важно(!) постарайтесь реализовать именно create database, а не только create table).  
drop_db_socialnet() - Удаление базы данных.  
create_table_socialpost() - Создание таблицы (после create_db_socialnet()).  
clear_table_socialpost() - Очистка таблицы.  
add_post(content_text, author, schedule_time, is_private) - Добавление новых данных (в моём случае, создание нового поста).  
search_post_by_author(temp_author) - Поиск по заранее выбранному (одному) текстовому не ключевому полю.  
update_post() - Обновление кортежа (строки таблицы).  
delete_post_by_author() - Удаление по заранее выбранному (одному) текстовому не ключевому полю.  

__2. Дополнительные функции (_прописано в SQL_):__  
create_user_socialnet() - Создание пользователя (помимо Администратора).  
drop_user_socialnet() - Удаление пользователя.  
see_all_posts() - Просмотр всех постов.  
change_likes_amount() - Изменение количества лайков (относится к обновлению).  

_(P.S. в окне Администратора не работала функция "Создать новую БД с таблицей")._
